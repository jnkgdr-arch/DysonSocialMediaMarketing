# Dyson Social Media Marketing Analysis

A static, GitHub Pages-ready executive dashboard for a Dyson social media marketing analysis.

## Files

- `index.html` — page structure and all provided analysis content.
- `styles.css` — dark gradient executive styling, responsive cards, navigation, and chart layout.
- `script.js` — responsive canvas charts and mobile navigation behavior.
- `assets/` — provided social media and campaign images used in their relevant sections.

## Run locally

Open `index.html` directly in a browser, or serve the folder locally:

```bash
python3 -m http.server 4173
```

Then visit `http://127.0.0.1:4173/`.

## Deploy on GitHub Pages

1. Push this repository to GitHub.
2. Open repository **Settings**.
3. Go to **Pages**.
4. Select the current branch and repository root as the source.
5. Save and open the generated GitHub Pages URL.
